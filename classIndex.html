EXecutioner API Documentation

Class Hierarchy

  • Exception
    • src.errors.CompilationError - represent compilation error of program
    • src.errors.MemoryOutError - represent memory error of program
    • src.errors.RunTimeError - represent runtime error of program
    • src.errors.TimeOutError - represent time out error of program
  • src.evaluate.Evaluation - Evaluate the given program against Testcases and score with Metrics
  • src.evaluate.TestCase - Represent a testcase
  • src.metric.base.BaseMetrics - base class of Metrics
    • src.metric.equal.Equal - a 0/1 metric, gives full score if equal
  • src.program.Program - Represent a program instance
  • src.sandbox.base_sandbox.SandBox - SandBox base class
    • src.sandbox.no_sandbox.NoSandBox - NoSandBox sandbox class
      • src.sandbox.firejail.FireJail - FireJail SandBox class
  • src.sandbox.base_sandbox.SandBox.CompiledProgram - compiled program class for file based sandboxing
  • src.sandbox.test.sandbox_tester.BaseTestSandBox - Base Tests for all sandboxes
    • src.sandbox.test.language_tester.CTestSandBox - Tests for C Language
      • src.sandbox.test.test_sandbox.TestFireJail - Tests for FireJail class
      • src.sandbox.test.test_sandbox.TestNoSandBox - Tests for NoSandBox class
    • src.sandbox.test.language_tester.PythonTestSandBox - Tests for Python Language
      • src.sandbox.test.test_sandbox.TestFireJail - Tests for FireJail class
      • src.sandbox.test.test_sandbox.TestNoSandBox - Tests for NoSandBox class
    • src.sandbox.test.sandbox_tester.SecureTestSandBox - Base Test file of all secure sandboxes
      • src.sandbox.test.test_sandbox.TestFireJail - Tests for FireJail class
  • src.settings.Settings - Represent the settings
  • src.test.test_settings.TestLoaders - Collection of tests to check the settings loaders
